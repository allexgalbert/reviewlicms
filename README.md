# Reviewli CMS

Fast and easy open source CMS.

## Requirements

- PHP 7.4+
- MySQL 8.0+
- Laravel 8.12+

## Installation

```sh
composer create-project --stability=dev allexgalbert/reviewlicms
```

## Features

- Categories
- Brands
- Features
- Websites
- Users
- Reviews
- Admins
